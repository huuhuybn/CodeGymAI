Phần 2

Trợ lý Lập trình Tiên tiến

Trình bày: Tech Expert

Các công cụ không thể thiếu sẽ thay đổi thói quen dùng IDE thường ngày và tăng tốc độ xử lý bài tập.

Cursor

Trình biên tập thế hệ mới (dựa trên VSCode). Sức mạnh của Composer giúp AI chỉnh sửa, đọc hiểu nhiều file cùng lúc để hoàn thành module chức năng.

Claude Code

Giải pháp xử lý dự án trực tiếp qua giao diện dòng lệnh (CLI). Bằng tư duy của Claude 3.7, nó tự do thao tác sửa lỗi trên toàn bộ thư mục.

Google DeepMind: Antigravity

  • Khả năng chủ động phân tích bài toán (Planning Mode).

Tài nguyên & Hạ tầng AI

Trình bày: DevOps / Cloud Specialist

1

Môi trường Nguồn Mở: Hugging Face

"GitHub của cộng đồng AI" - Hướng dẫn tải Models và Datasets phục vụ việc build sản phẩm.

2

Triển khai Cloud GPU: Thuê ở đâu?

RunPod (Quốc tế): Cực kỳ dễ setup template Container (PyTorch). Chi phí linh hoạt theo giờ, rất rẻ để thực hành cá nhân.
FPT AI Factory (Nội địa): Dòng máy chủ sức mạnh cao (NVIDIA H100, A100). Đảm bảo chuẩn bảo mật dữ liệu doanh nghiệp và pháp lý Việt Nam.

Phần 5

Hands-on: Tự Xây Dựng AI Service

Trình bày: Software Architect

Thực hành Demo Flow:

  1. Kéo mô hình (Ví dụ: Llama 3) từ kho Hugging Face.
  2. Đẩy script và setup môi trường lên RunPod hoặc FPT Cloud GPU.
  3. Viết nhanh một API bằng Python (FastAPI/Flask) bọc lại model để tạo endpoint giao tiếp.
  4. Tạo ứng dụng Frontend siêu nhỏ để gọi API này, chứng minh quy trình đưa AI vào app thực tế.
